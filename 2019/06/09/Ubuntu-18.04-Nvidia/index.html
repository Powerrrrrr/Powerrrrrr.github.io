Ubuntu 18.04 Install Nvidia driver

Posted on 2019-06-09

When I was installing the Nvidia driver for the laptop with Ubuntu 18.04 system, I got caught with many problems.

And after a lot of trials to solve all the matters and many rounds of install-reinstall, I successfully installed it.

Installation

First I would to share how I install the driver.

Preparation

Download run file

I chose the .run file to install the driver. You can get the correct run file on the official site of Nvidia drivers.

Download the correct run file that matches your device.

Remember to change the mode of the .run file to a executable file.

Disable Secure Boot

Enter the BIOS before the computer start and disable Secure Boot mode.

Different computers require different methods to enter the BIOS Interface. Maybe you can Google to check how to enter BIOS.

Disable the nouveau driver

You need to disable the nouveau driver ensure that the Nvidia driver could be installed.
